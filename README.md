ClubSphere is a centralized college club management platform built during a hackathon to solve the problem of scattered club information and unstructured recruitment processes in colleges.

It enables students to discover, explore, and apply to clubs easily, while providing a secure admin dashboard to manage applications and recruitment cycles.
